## Overview
Analyzed global e-commerce customer behavior using SQL and Excel to identify revenue drivers, customer segmentation, retention patterns, and purchasing behavior.

## Tools
- SQL / MySQL
- Excel
- Pivot Tables
- Data Visualization

## Key Insights
- Premium price segment is the primary revenue driver.
- Customers aged 25–60 contribute the majority of revenue.
- Retention has limited impact on average order value.
- Demand is stable across weekdays and regions.
