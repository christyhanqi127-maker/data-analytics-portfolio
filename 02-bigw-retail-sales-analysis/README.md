# Big W Retail Sales Analysis

## 🔗 Quick Access
- 📓 [View Full Analysis Notebook](./bigw.ipynb)

## Overview
Analyzed Big W retail sales data to identify sales drivers, media investment impact, customer segments, and store expansion opportunities.

## Tools
- Python
- Excel
- Machine Learning
- LightGBM

## Key Insights
- Media investment and local population are important revenue drivers.
- LightGBM achieved the best predictive performance.
- Store location and competitor proximity influence sales performance.
- Customer demographics help guide marketing and expansion strategy.
