# CPI Forecasting and Time Series Analysis

## Overview
Forecasted Consumer Price Index trends using statistical and machine learning models to analyze inflation patterns and model performance.

## Tools
- Python
- Time Series Analysis
- ARIMA / SARIMA
- LSTM
- Model Evaluation

## Key Insights
- CPI shows long-term upward trend and quarterly seasonality.
- SARIMA provides strong interpretability.
- LSTM achieves strong predictive accuracy.
- Model selection requires balancing accuracy and interpretability.
