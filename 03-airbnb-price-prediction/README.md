# Airbnb Price Prediction

## 🔗 Quick Access
- 📓 [View Full Analysis Notebook](./airbnb_price_prediction.ipynb)

## Overview
Built machine learning models to predict Airbnb listing prices and generate market insights for hosts, investors, and property managers.

## Tools
- Python
- Feature Engineering
- Machine Learning
- LightGBM
- Lasso Regression
- Decision Tree

## Key Insights
- Property attributes such as amenities, bedrooms, and capacity influence price.
- Location significantly affects pricing.
- LightGBM achieved the strongest predictive performance.
- Superhost behavior provides useful market insight.
